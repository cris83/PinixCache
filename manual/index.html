Welcome to the PINIX Cache Docs!¶

이 문서는 PINIX 환경설정에 관한 내용 입니다. ( This documents specify the PINIX Configurations. )

Introduction¶

PINIX는 운영자의 편의와 정확성을 위해 기본적으로 웹 환경설정을 지원하고 있습니다. 웹을 통해서 생성된 설정정보는 서버의 특정 위치에 JSON 파일로 생성됩니다. 이 문서에서는 JSON 파일의 구성 및 각 항목별 내용에 대해 상세 기술되어 있습니다.
